# Financial Indicators Reference (TA-Lib)
A brief technical reference for calculating and plotting basic indicators using TA-Lib and yfinance.

## Features
Moving Averages: SMA (21) and EMA (55).

## Volatility: Bollinger Bands (20).

## Momentum: RSI (14).

## Tech Stack
Python (developed in PyCharm).

## Libraries: talib, yfinance, matplotlib.

### Quick Start
Bash
pip install yfinance matplotlib talib-binary
python main.py
